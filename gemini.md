## Deprecated resouces:
The following objects / access are discouraged. If there is a SDK equivalent to achieve the same, it must be used.
- The `W`/`Waze` object
- The `OL`/`OpenLayers` objects. See Openlayers 2: https://web.archive.org/web/20200701150732/http://dev.openlayers.org/releases/OpenLayers-2.13.1/doc/apidocs/files/OpenLayers-js.html
- The `I18n` object, contains localized strings.
